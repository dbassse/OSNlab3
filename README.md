# 🐻‍❄️Сверточные нейронные сети🐻‍❄️

 Цель проекта — достижение точности классификации не ниже **93%** на проверочной выборке.

## 📌 Описание задачи
В рамках данного проекта была решена задача многоклассовой классификации изображений. Основная сложность заключалась в необходимости высокой точности при ограниченном наборе данных, что потребовало применения современных техник обучения глубоких нейронных сетей.

## 🛠 Используемые технологии
* **Python 3.10+**
* **TensorFlow / Keras** (построение и обучение модели)
* **Matplotlib / PIL** (визуализация данных и графиков)
* **Google Colab** (среда для обучения с использованием GPU)

## 🏗 Архитектура решения
Для достижения целевых показателей был выбран метод **Transfer Learning** (перенос обучения) с использованием готовой архитектуры из модуля `tf.keras.applications`.

### Основные этапы:
1. **Подготовка данных**: 
   - Загрузка изображений с автоматическим разделением на `train` и `validation` выборки.
   - Масштабирование изображений.
   - Применение слоев **аугментации** (RandomFlip, RandomRotation, RandomZoom) для повышения обобщающей способности модели.
2. **Базовая модель**: Использование предобученной сети.
3. **Методика обучения**:
   - Заморозка весов базовой модели и обучение только финальных полносвязных слоев (головы).
   - Частичная разморозка глубоких слоев базовой модели и дообучение на очень низком Learning Rate для тонкой настройки под специфику марок авто.


### График точности
В ходе обучения отслеживались метрики Accuracy на тренировочном и валидационном наборах данных.
